# Omarchy Customisations

After a fresh install of Omarchy apply my customisations to set it up how I like it. 

## Setup 

- Clone this repo
- CD into the customisations dir
- run the master-install.sh 
    `./master-install.sh`

- Setup git cli auth to create and upload ssh key to Github
    `gh auth login`  -- follow the prompts
- If you want to push to dotfiles repo change the remote URL to use ssh
    `git remote set-url origin git@github.com/tbusby/dotfiles.git`
- Setup Firefox
    Login to Firefox profile
- Setup Thunderbird 
    Login to gmail 
- Setup Obsidian
    Login to Obsidian Sync
    Input pass to unencrypt notes 
    Settings > Sync > Activate Community Plugins && Sync Community Plugins
    Settings > Plugins > Enable Community Plugins 
    Settings > Plugins > Sync Plugins 
    Settings > Plugins > Update Plugins 
    Move Calendar to left bar
